# דף HTML עם אנימציה

איך להריץ:
- לפתוח את `index.html` בדפדפן (Double click).

קבצים:
- `index.html` – העמוד
- `style.css` – העיצוב/אנימציות
- `script.js` – טייפרייטר + אפקט קנבס
- `assets/background.png` – תמונת רקע

הערה:
שמתי דמות "כועסת ומוגזמת" בסגנון איורי כללי (לא קריקטורה שמכוונת לקבוצה דתית מסוימת).
